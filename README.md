# 🐧 Linux System Monitor (C / Low-Level)

A lightweight system resource monitor written in **C** that interacts directly with the Linux kernel via the `/proc` filesystem. This project demonstrates proficiency in systems programming and hardware-software interfacing.

## 🛠 Features
*   **Memory Tracking**: Real-time extraction of RAM usage from `/proc/meminfo`.
*   **Low-Level Access**: No heavy libraries, just pure C and kernel interfaces.
*   **Performance**: Extremely low overhead and memory footprint.
